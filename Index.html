⚪ ยังไม่เชื่อม Sheet
1. เปิด Google Sheet ใหม่ → เมนู Extensions › Apps Script
2. วางโค้ดไฟล์ Code.gs (ผมแนบให้ในแชต) ทับทั้งหมด แล้วบันทึก
3. กด Deploy › New deployment › Web app · Execute as = Me · Who has access = Anyone
4. คัดลอก URL ที่ลงท้าย /exec มาวางช่องด้านบน แล้วกด บันทึกลิงก์
5. กด ส่งทั้งหมดขึ้น Sheet 1 ครั้งเพื่อเริ่มต้นฐานข้อมูล จากนั้นทุกการเปลี่ยนแปลงจะบันทึกขึ้น Sheet อัตโนมัติ
