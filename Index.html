วิธีตั้งค่า Google Sheet
1. เปิด Google Sheet ใหม่ → Extensions › Apps Script
2. วางโค้ด Code.gs ทับทั้งหมด แล้วบันทึก
3. Deploy › New deployment › Web app
  Execute as = Me · Who has access = Anyone
4. คัดลอก URL ลงท้าย /exec วางช่อง Apps Script URL
5. กด บันทึกลิงก์ แล้วเพิ่ม/แก้ไขข้อมูลผ่านแอป ระบบจะบันทึกลง Sheet ทีละรายการ
